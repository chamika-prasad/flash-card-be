# Getting Started with Flash-Card-Be App

## Instructions for Setting Up the Project

### Node.js Version Requirement 
Ensure that your system has Node.js version `v18.17.1` installed.

### Clone the Repository 
Execute the following command to clone the project repository
```bash
git clone https://github.com/chamika-prasad/flash-card-be.git
```

### Install Dependencies
Navigate to the project directory and install the required packages by running:

```bash
npm install
```

### Create Database
Create Database by running quiries inside `Flash_App_DB` file


### Create Amin User
Create Admin user using the following command:

```bash
npm run init-admin
```

### Run the Project
Start the application using the following command:

```bash
npm run dev
```

### Run Tests
Run the application Tests using following command:

```bash
npm test
```
